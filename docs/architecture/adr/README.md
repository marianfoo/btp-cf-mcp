# Architecture Decision Records

Accepted decisions belong here as individual ADR files.

Naming:

```text
0001-short-kebab-title.md
0002-next-decision.md
```

Each ADR should be short and include:

- Status
- Date
- Context
- Decision
- Consequences
- Supersedes / Superseded by, when relevant

Long research, transcripts, and experiments belong in `docs/research/` or `docs/operations/` and should be linked from
the ADR rather than pasted into it.
